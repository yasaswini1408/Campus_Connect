Issue with Dijkstra's Algorithm implementation Resolved
Asked by: John Smith (3rd Year, ECE) | Topic: Data Structures
My implementation of Dijkstra's seems to enter an infinite loop on graphs with negative weights. Is this expected behavior? What algorithm should I use instead?

Your Resolution:

Yes, Dijkstra's algorithm fails with negative edge weights as it assumes shortest paths are found greedily. You should use the **Bellman-Ford algorithm** instead, which is designed to handle negative weights and detect negative cycles.
